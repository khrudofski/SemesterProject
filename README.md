# Supply the Spark
A web application for a fictonal charity that supports registration for 5k races

# Overview
Supply the Spark is a charity race management web application designed to help users register for 5k events, create teams, and support charitable causes through community-driven races. The platform includes public features for participants and administrative tools for managing runners.

# Features
- User account registration and login
- Race registration
- Team creation
- Charity information
- User dashboard for viewing race information
- Admin dashboard for managing runners
- Responsive design for desktop and mobile devices

# Goals
This project was created to practice and demonstrate frontend web development with React, responsive UI/UX design, state management, and real-world application structure. Future improvements that could be added are email confirmations, enhanced accessibility features, and dark mode support.
